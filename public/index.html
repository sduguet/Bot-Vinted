🔑 Cookie de session manuel
Le cookie automatique est bloqué (Datadome). Récupère-en un depuis ton navigateur :
  1. Ouvre https://www.vinted.fr et connecte-toi
  2. Appuie sur F12 → onglet Réseau (Network)
  3. Dans le champ filtre (filter), tape : catalog/items
  4. Sur Vinted, lance une recherche (ex: "7 Wonders") → Tu dois voir apparaître une requête dans l'onglet Réseau
  5. Clique sur cette requête → onglet En-têtes (Headers)
  6. Dans En-têtes de requête (Request Headers), copie la valeur complète du champ Cookie (très longue chaîne)
  7. Et colle-la dans l'input en dessous 👍
⚠ Le cookie expire après quelques heures — répète l'opération si l'erreur revient.
