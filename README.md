# 🤖 VintedBot

Thèmes : `🔍 Surveillance` `🛒 Vinted` `⚡ Alertes` `🤖 Bot`

VintedBot est un bot de surveillance d'annonces Vinted.

Configurez vos alertes en quelques clics, définissez vos mots-clés et fourchettes de prix, puis laissez le bot scanner Vinted à intervalle régulier. Dès qu'une nouvelle annonce correspond à vos critères, elle s'affiche dans l'interface avec photo, prix et lien direct vers la fiche. Fini les bonnes affaires ratées.

📃 Fonctionnalités :

* Alertes multiples avec mots-clés et filtres de prix indépendants.
* Scan automatique à intervalle personnalisable.
* Nombre de résultats par alerte paramétrable.
* Interface entièrement dans le navigateur.
